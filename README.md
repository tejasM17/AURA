### How to Contribute

1. **Fork** this repository (click the **Fork** button on top right)
2. **Clone** your forked repo to your computer:
---
```
   git clone https://github.com/YOUR_USERNAME/AURA.git
   cd AURA
```
3. Create a new branch (Never commit directly to main) 
```
   checkout -b your-branch-name
```
4. Make your changes.
5. Remove cache before committing (especially .idea folder)
```
   git add .
   git rm -r --cached .idea
```
6. Commit your changes.
```
   git commit -m "Your meaningful commit message"
```
7. Push to your branch with PR
```
   git push origin your-branch-name
```
